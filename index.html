Function Tutorial

What is a function?

A function is a related group of JavaScript statements that are executed as a single unit.

Why is a function useful?

Functions can help organize your code. They are like tiny programs.

There are two types of functions: Named and Anonymous. Named functions can be called multiple times, anonymous functions only once where they are placed.
Functions must be written a certain way, called a function definition. First you must let the script know we are creating a function by typing the word function.

Second we must give the function a name if it is of the named type so we can refrence it. A function can be called anything, but it is best to name it something descriptive. The preferred naming scheme is usually two words, with the first all lowercase and the second capitalizing the first letter.

		//named function definition

Third we place a set of parenthesis after the name. Inside the parenthesis is where we place parameters, which are variables that are declared and used within the function.
You can have multiple parameters in the parenthesis seperated by commas.

		//named function definition

Fourth we place a set of braces after the parameters, which are called function braces. The braces act as a container so the script knows that everything inside of them belong to that function. These are what do the actual work of the function.

		//named function definition

Fifth and finally we add function statements inside of the function braces. These statements have what is called local scope, they can only be used within this function. Function statements must end with a semicolon.
